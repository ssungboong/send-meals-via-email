# send-meals-via-email

> "쳇, 또 막혀있군."

그는 몇 번이고 [http://front.cjfreshmeal.co.kr](http://front.cjfreshmeal.co.kr)에 접속을 시도했지만 그의 요청은 클라이언트가 지정한 Default gateway를 타고 인터넷 세상으로 나갈 수 없었다. 공허한 **Request timed out**이라는 메시지만 되돌아올 뿐, 어떠한 반응도 없었다.

하지만 그의 인생은 이 단순명료한 Javascript 파일을 Google Drive에 업로드되어 있는 아무 시트에나 추가한 뒤로 달라졌다.
